## Quantum BodyWorks Chatbot

This repository contains the code for tha RAG based chatbot that answers users queries about services, policies, and appointments.

> [!IMPORTANT]
> The goal for this project is to develop a RAG based system that can serve as a query chatbot for any service. The agent should ask the users to upload their documents that contain information about the policies, terms and conditions, services, etc. and should be able to answer questions about them irrespective of the nature of the service.
> I.E the goal is a generalized RAG driven query chatbot.
